  • Returns

  • Exceptions thrown

    Error Nodes must be on the same graph

  • Create a directed edge

    // Graph definition
let graph = new Graph()
let a = graph.add(new Node()), b = graph.add(new Node())
graph.edge(a, b, 1, null)
// Going from a to b cost 1 whereas it isn't possible to go from b to a
graph.adjacent(a, b) //True
graph.adjacent(b, a) //False

fromArray (array, …options) → {Graph}

lowlight.astar.js (line 264)

Utilitary method to create graph and subgraph from 2D Arrays.

  • Arguments

    array Array.<Object> Array
    options Object Options, specifying multiple options object will create subgraph with reused nodes
    order
    Default : "yx"     		String Array access order, accepted values are "xy" and "yx"
    cost
    Default : Graph.fromArray.cost     		Object Cost function
    torus
    Default : false     		Boolean If enabled, map will be treated as a torus
    diagonals
    Default : false     		Boolean If enabled, diagonals movements are allowed
    cutting
    Default : false     		Boolean If enabled, diagonals movements between two blocking cells are allowed

  • Returns

    Graph Graph generated from array

  • Creating a graph from an Array structure

    // Assume with have this map
let map = [
     [0, 1, 0],
     [0, 2, 1],
     [0, 0, 0]
]

// Define options
let options = { }

// To access a member of our array, we use following notation : map[y][x]
options.order = "yx"

// Define cost function
// Return null to prevent links between two squares
options.cost = (a, b) => { return b >= 0 ? b : null }

// Tell if map is torus (map wrapper on itself)
options.torus = true

// Tell if squares must be linked diagonally
options.diagonals = true
// Tell if squares must be linked diagonally between two blocking squares
options.cutting = false

// Start by defining options
let graph = Graph.fromArray(map, options)

id (id) → {Number}

lowlight.astar.js (line 54) 

Convert an object to a node id.
This method should be overridden when you create a graph from an existing structure.

  • Arguments

    id Object Id to convert

  • Returns

    Number Id

  • Overriding Graph.id method

    // Graph definition
let graph = new Graph()
let node = graph.add(new Node(3))
graph.id = (coordinates) => { return coordinates.x % 2 + coordinates.y * 2 }
// Nodes can now be accessed with id others than numbers
graph.id({x:1, y:1})
