edge (a, b, cost) → {Graph}

lowlight.astar.js (line 127)

Create an edge between two vertices. Set NaN on one edge cost to mark it as a directed edge. Set NaN on both edge to remove edge.

  • Arguments

    a Vertex Vertex A
    b Vertex Vertex B
    cost Object Edge costs for vertices A and B
    ab
    Default : 1     		Number Edge cost from vertex A to vertex B
    ba
    Default : 1     		Number Edge cost from vertex B to vertex A

  • Returns

    Graph Instance

  • Exceptions thrown

    Error Vertices must be on the same graph

  • Create a directed edge

    //Graph definition
  const graph = new Graph()
  const a = graph.add(new Vertex()), b = graph.add(new Vertex())
  graph.edge(a, b, {ab:1, ba:NaN})
//Going from A to B cost 1 whereas it isn't possible to go from B to A
  graph.adjacent(a, b) //True
  graph.adjacent(b, a) //False

  • Create a directed edge

    //Remove edge
  graph.edge(a, b, {ab:NaN, ba:NaN})
